In this task, you will use your knowledge about language (and common sense) to determine what is the most suitable sentiment to describe the given text.

Task Definition

  • Carefully read the given text.
  • The text is an email sent by employees and need to be addressed accordingly.
  • Sometimes, some email headers might appear in the text, you may ignore them when evaluating the text.
  • When you don't find any suitable option, you should choose the 'other' option and write a sentiment to your choice, please make sure not to write more than 2 words to describe it.
Make sure to read all the examples and their explanations to get a notion of the different cases
